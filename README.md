# Virtual Impact

Frontend-first project skeleton focused on clean organization and scalable feature grouping.

## Top-level structure
- `frontend/app/`: route-level pages and sections
- `frontend/components/`: shared UI and layout components
- `frontend/lib/`: helpers, constants, and utility logic
- `frontend/styles/`: tokens, animations, and global styles
- `frontend/public/`: static assets (images, icons)
