# Public image placement

Use this folder for all static image assets.

## Where to place files
- `public/images/home/`: homepage and hero visuals
- `public/images/services/`: service page visuals
- `public/images/shared/`: reusable graphics used across multiple pages
- `public/images/branding/`: logo, mark, and brand kit assets
- `public/icons/`: icon files (SVG/PNG)

## How to reference in React/Next
- Example: `/images/home/hero-bg.jpg`
- Example: `/icons/arrow-right.svg`
